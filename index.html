Github

How to create a repo

Step 1

Access your GitHub account and click on the onto repositories along the top

Step 2

Click the new button

Step 3

Name the repo and add a brief description and remember to click add a README file, and that will create the repo, next up you want to clone the repo onto your vs code (remember not to make changes on the github website - always make the changes in vs code).

How to clone a repo

Step 1

Create the repo on Github then click into it. It should look like the image

Step 2

Click on the green code button in the top right and copy the link to the repo

Step 3

using file explorer, navigate to the folder you want the repo to be in, then right click and select more options then git bash

Step 4

type git clone and paste the link you copied from the repo, note that you cannot control + v to paste in git bash, you have to right click and select paste.

Step 5

Finally, open vs code and open the folder you just cloned, and you're set.
