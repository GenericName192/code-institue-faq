Github

How to create a repo

Step 1

Access your GitHub account and click on the onto repositories along the top

Step 2

Click the new button

Step 3

Name the repo and add a brief description and remember to click add a README file, and that will create the repo, next up you want to clone the repo onto your vs code (remember not to make changes on the github website - always make the changes in vs code).
